# 流萤研习室 — 考研备考计划工具

基于浏览器端存储的考研每日学习计划与番茄钟工具，纯前端实现（HTML/CSS/JS），配合 Python 后端实现数据持久化。

## 功能

- **每日任务打卡** — 5月11日至5月31日三周考研计划，涵盖数学（概率论、线代、高数）、英语（六级）、专业课（机械设计、机械原理）
- **番茄钟计时** — 随机40-70分钟学习时段 + 5分钟休息，支持选择具体学习任务
- **番茄钟统计** — 单独窗口展示每日学习时长分布、累计趋势曲线、周学习量对比
- **学习热力图** — 21天打卡情况可视化（颜色深浅 + 遗漏标记）
- **周进度对比** — 三个周完成度柱状图对比
- **周报** — 按周汇总各科完成数据
- **光萤之树** — 随总进度百分比成长的视觉反馈
- **成就系统** — 9个阶段性成就徽章
- **离线可用** — 无服务器时自动降级为 localStorage 存储

## 使用方法

### 启动服务器

```bash
python server.py
```

浏览器访问 `http://127.0.0.1:8765/`

### 纯本地使用

直接用浏览器打开 `五月每日计划.html`，数据保存在 localStorage 中。
