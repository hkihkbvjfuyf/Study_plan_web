# 流萤研习室 — 考研备考计划工具

基于浏览器端存储的考研每日学习计划与番茄钟工具，配合 Python 后端实现数据持久化。

## 功能

### 学习管理
- **每日任务打卡** — 5月11日至5月31日三周考研计划，涵盖数学（概率论、线代、高数）、英语（六级）、专业课
- **番茄钟计时** — 随机40-70分钟学习时段 + 5分钟休息，支持选择具体学习任务
- **番茄钟统计** — 独立窗口展示每日学习时长分布、累计趋势曲线、周学习量对比，时间段格式显示
- **学习热力图** — 21天打卡情况可视化（颜色深浅 + 遗漏标记）
- **周进度对比** — 三个周完成度柱状图对比
- **周报** — 按周汇总各科完成数据
- **光萤之树** — 随总进度百分比成长的视觉反馈
- **成就系统** — 9个阶段性成就徽章

### 日记系统
- **选项式日记** — 8个维度记录每日状态，无需打字：学习状态、心情、专注程度、学习时长、今日困难、身体状态、学习方式、入睡/起床时间、明日计划
- **多选支持** — 今日困难和学习方式可多选
- **每日总结** — 综合任务完成率、番茄钟数据、日记记录、专注监控数据，生成评分和评价文字
- **手动刷新** — 总结支持手动刷新，凌晨4点自动生成昨日首次总结
- **历史回顾** — 按日期查看过往日记和总结

### AI 聊天助手
- **流萤人格** — 模仿《崩坏：星穹铁道》角色"流萤"的温柔陪伴风格
- **计划修改** — 直接在聊天中讨论调整任务、生成后续计划
- **自动刷新** — AI 修改计划后页面自动更新
- **上下文感知** — 聊天自动附带当前计划数据和完成进度

### 专注监控
- **窗口活动追踪** — 后台检测当前活动窗口标题，识别摸鱼行为
- **智能过滤** — 黑白名单机制，学习相关关键词自动排除
- **实时仪表板** — 专注时长、摸鱼时长、专注率、摸鱼次数实时展示
- **数据分析弹窗** — 每小时时间线图、近7日对比、摸鱼来源排行
- **活动日志** — 按时间线展示全天活动记录
- **黑白名单管理** — 可视化编辑关键词，修改即时生效

### 时间规则
- **凌晨4点分界** — 0:00-3:59 的番茄钟记录和日记归属于前一天

## 使用方法

### 启动服务器

```bash
python server.py
```

浏览器访问 `http://127.0.0.1:8765/`

### 启动专注监控（可选）

```bash
python focus_monitor.py
```

### 纯本地使用

直接用浏览器打开 `五月每日计划.html`，数据保存在 localStorage 中。（专注监控不可用）

## 数据 API

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/state` | GET | 获取全部状态（含专注数据） |
| `/api/state` | POST | 保存状态 |
| `/api/focus` | GET | 获取专注监控数据 |
| `/api/focus_config` | GET/POST | 读写黑白名单配置 |
| `/api/chat` | POST | AI 聊天 — 通过 Claude Code CLI 中转 |

## 技术栈

- 前端：原生 HTML / CSS / JavaScript，Canvas 图表
- 后端：Python `http.server`，JSON 文件存储
- 监控：pywin32（窗口检测）+ JSON 持久化
- 无外部 JS 依赖（除 Google Fonts）
